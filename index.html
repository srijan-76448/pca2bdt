MongoDB Basic Operations

Command Description
use <database_name> Create Database/Switch to Database
db.createCollection("<collection_name>") Create Collection
db.<collection_name>.insertOne(<document>) Insert One Document
db.<collection_name>.insertMany([<document1>, <document2>, ...]) Insert Many Documents
db.<collection_name>.find(<query>, <projection>) Find Documents
db.<collection_name>.updateOne(<query>, <update>, <options>) Update One Document
db.<collection_name>.updateMany(<query>, <update>, <options>) Update Many Documents
db.<collection_name>.deleteOne(<query>, <options>) Delete One Document
db.<collection_name>.deleteMany(<query>, <options>) Delete Many Documents
db.<collection_name>.drop() Drop Collection
db.dropDatabase() Drop Database
show collections Show Collections
show dbs Show Databases

Hadoop File Management Commands

Command Description
hadoop fs -mkdir <paths> Create Directory
hadoop fs -ls <args> List Directory Contents
hadoop fs -put <localsrc> <HDFS_dest_Path> Upload File
hadoop fs -get <hdfs_src> <localdst> Download File
hadoop fs -cat <path [filename]> See File Contents
hadoop fs -cp <source> <dest> Copy File
hadoop fs -copyFromLocal <localsrc> URI Copy From Local to HDFS
hadoop fs -copyToLocal [-ignorecrc) (-crc] URI <localdst> Copy From HDFS to Local
hadoop fs -mv <src> <dest> Move File
hadoop fs -rm <arg> Remove File/Directory
hadoop fs -rmr <arg> Remove Directory Recursively
hadoop fs -tail <path [filename ]> Display Last Lines
hadoop fs -du <path> Display File Length
